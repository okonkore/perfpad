# ユニバーサル基板レイアウトツール 仕様書

## 目的

ブラウザだけでユニバーサル基板の部品配置、配線、カット位置を設計できるレイアウトツール。

ユニバーサル基板では、部品のリードを「どの穴からどの穴へ挿すか」が重要になるため、抵抗やコンデンサなどの素子は開始穴から終了穴までドラッグして穴間隔を決められることを重視する。

## 実行方法

- `index.html` をブラウザで開く。
- ビルドやサーバー起動は不要。
- レイアウトはブラウザの `localStorage` に自動保存される。

## 画面構成

- 上部バー
  - Undo
  - Redo
  - PNG 書き出し
  - JSON 書き出し
  - JSON 読み込み
  - 全削除
- 左パネル
  - 基板サイズ設定
  - ズーム
  - 座標表示切替
  - 表示設定
  - 掴む対象切替
  - 配置ツール
  - 配線色
  - 部品面 / はんだ面切替
- 中央キャンバス
  - 基板、穴、部品、配線、選択範囲を描画する。
- 右パネル
  - 未選択時は集計情報を表示する。
  - 単体選択時はインスペクタを表示する。
  - 複数選択時は選択数と内訳、削除操作を表示する。

## 基板仕様

- 穴は格子状に配置する。
- 初期サイズは `30 x 20` 穴。
- 穴ピッチは表示上 `2.54 mm` とする。
- 横方向は `8 - 80` 穴、縦方向は `8 - 60` 穴の範囲で変更できる。
- 基板サイズ変更時、既存アイテムは基板範囲内に収まるように丸める。
- 基板色は緑、茶、青、黒から選択できる。
- 穴設定画面で、各穴を有効 / 無効に切り替えられる。
- 無効穴は銅ランドを描画せず、暗いマークで表示する。
- 無効穴には新規の部品ピン、配線点、カットマークを配置できない。
- 無効穴を追加しても、既存アイテムは自動削除しない。

## ツール仕様

### 選択

- アイテムをクリックすると単体選択する。
- 空白からドラッグすると範囲選択する。
- 範囲に触れた部品、配線、カットマークを選択対象にする。
- 複数選択中に選択済みアイテムをドラッグすると、選択セット全体を穴単位で移動する。
- グループ移動は基板外に出ない範囲に制限する。
- 素子を移動する場合、移動開始時点でその素子のピンと同じ穴にある未選択の配線点は、素子と同じ差分で追従して接続を保つ。
- 配線の途中が素子ピンの穴を通過しているだけで配線点がない場合は、自動では追従させない。
- 選択中の配線は従来どおり配線全体を移動し、素子追従による二重移動はしない。
- 選択済み配線が単体の場合、配線の端点をドラッグして編集できる。
- 掴む対象は `自動`、`素子`、`配線` から切り替えられる。
- `自動` では、素子本体と配線が重なった場所では素子本体を優先し、選択済み配線の端点は端点編集として優先する。
- `素子` では配線を掴まず、抵抗、コンデンサ、LED、IC、ピンヘッダ、ラベル、カットマークだけを選択対象にする。
- `配線` では素子やカットマークを掴まず、配線の端点または線分だけを選択対象にする。
- 掴む対象の切替は編集データには保存せず、現在の操作補助として扱う。

### ズーム

- ズームスライダーで表示倍率を変更できる。
- iPhone / iPad などのタッチ端末では、キャンバス上のピンチイン / ピンチアウトで表示倍率を変更できる。
- ピンチ操作中はページ全体をスクロールまたはブラウザズームさせず、キャンバス操作として扱う。
- デスクトップや DevTools 検証時は、キャンバス上で `Ctrl` / `Cmd` / `Option` を押しながらホイールまたはトラックパッド操作をすると表示倍率を変更できる。
- 表示倍率は `0.7 - 2.2` の範囲に制限する。

### 表示設定

- 配線、素子、素子名ラベルはそれぞれ独立して `ON`、`OFF`、`半透明` を切り替えられる。
- 初期表示は `配線 ON`、`素子 半透明`、`名前 ON` とする。
- 配線表示は、部品面配線とはんだ面配線の両方に適用する。
- 素子表示は、抵抗、コンデンサ、LED、IC、ピンヘッダ、テキストラベル、カットマークに適用する。
- 素子名は基板上で読みやすいよう、素子本体とは別に水平な高コントラストラベルとして描画する。
- 素子名ラベルの `半透明` は、ラベルの背景と文字をまとめて薄く表示する。
- `OFF` にしたカテゴリは表示しないだけでなく、クリック選択や範囲選択の対象からも外す。
- 表示設定は編集データには保存せず、見え方だけを切り替える。

### 配線

- 配線ツールで穴をクリックして折れ線の点を追加する。
- 右クリック、ダブルクリック、Enter、または画面上の確定ボタンで配線を確定する。
- 取消ボタンまたは Escape で配線ドラフトを破棄する。
- 配線色は 6 色から選択できる。
- 部品面配線とはんだ面配線を切り替えられる。
- はんだ面配線は破線かつ半透明で表示する。
- 配線確定後、または配線の移動 / 端点編集後に、同じ面・同じ色・同じ太さの配線の端点同士が同じ穴で接触している場合は、1 本の折れ線配線へ自動統合する。
- 自動統合は端点同士のみを対象にし、配線の途中に別配線の端点が触れる T 字接続は別アイテムのまま扱う。
- 自動統合後は、統合された配線全体を 1 回のドラッグで移動できる。

### 接続リスト

- インスペクタの未選択表示では、各素子の各ピンが何に接続されているかを一覧表示する。
- 素子を単体選択した場合は、その素子の接続だけをインスペクタ内に表示する。
- 複数選択時は、選択中の素子の接続だけを表示する。
- 接続判定は、同じ穴にあるピンと、配線でつながった穴を同じネットとして扱う。
- 配線が水平、垂直、または整数座標の穴を通る斜め線の場合、線が通過する穴も同じネットに含める。
- 無効穴はネットに含めず、配線が無効穴を通過する場合はそこでネットを分断する。
- 抵抗、コンデンサ、LED、IC、ピンヘッダの各ピンは接続リストの対象にする。
- ラベル、カットマーク、配線アイテムそのものは接続リストの素子としては表示しない。
- 抵抗やコンデンサなどの素子内部は短絡として扱わず、各ピンは別々の接続点として扱う。
- 同じネットに他の素子ピンがある場合は `U1-3` のように接続先を表示する。
- 同じネットに配線だけがあり他の素子ピンがない場合は `配線のみ`、どの配線にもつながっていない場合は `未接続` と表示する。
- 接続リストは Markdown ファイルとして書き出せる。
- Markdown には部品の種類、外観種類、値、各ピンの接続先を含める。
- 部品リストは Markdown ファイルとして書き出せる。
- 部品リストは購入時に見やすいよう、同じ種類、外観種類、値、ピン数の部品をまとめ、数量と使用部品名を含める。
- 部品リストは Google スプレッドシートなどで読み込める CSV ファイルとしても書き出せる。
- ログインなしで、複数のレイアウトを名前付きでサーバー側に保存できる。
- サーバー保存は Deno KV を使い、保存済みレイアウトの一覧、読み込み、削除ができる。

### 抵抗

- 開始穴から終了穴までドラッグして配置する。
- ドラッグ方向に応じて水平または垂直に配置する。
- 表示は細い軸型抵抗らしく、スリムな円筒ボディ、細いリード、カラー帯で描画する。
- 既定の 3 穴間隔では、ボディ長を約 2 穴強、太さを約 0.5 穴として表示する。
- 外観は茶色カーボン、青い金属皮膜、緑の金属皮膜から選択できる。
- 保存値は `length`。
- クリックのみの場合は既定で 3 穴間隔。
- インスペクタで穴間隔、外観、名前、値、回転を編集できる。

### コンデンサ

- 開始穴から終了穴までドラッグして配置する。
- ドラッグ方向に応じて水平または垂直に配置する。
- 外観は積層セラミック、電解、フィルムから選択できる。
- 電解コンデンサは真上から見た円形の缶として描画する。
- 保存値は `spacing`。
- クリックのみの場合は既定で 2 穴間隔。
- インスペクタで穴間隔、外観、名前、値、回転を編集できる。

### LED

- 開始穴から終了穴までドラッグして配置する。
- ドラッグ方向に応じて水平または垂直に配置する。
- 保存値は `spacing`。
- クリックのみの場合は既定で 2 穴間隔。
- インスペクタで穴間隔、名前、値、色、回転を編集できる。

### DIP IC

- クリックした穴を基準に配置する。
- 初期値は 8 ピン、幅 3 穴。
- 表示は DIP IC らしい黒い長方形ボディと、左右のピン穴へ伸びる銀色のリードで描画する。
- インスペクタでピン数、幅、名前、値、回転を編集できる。

### ピンヘッダ

- 開始穴から終了穴までドラッグして配置する。
- 終了穴を最後のピン位置として扱う。
- 保存値は `count`。
- クリックのみの場合は 4 ピン。
- インスペクタでピン数、名前、値、回転を編集できる。

### ラベル

- クリックした穴に文字を配置する。
- インスペクタで文字と色を編集できる。

### カットマーク

- クリックした穴にカットマークを配置する。
- すでに同じ穴にカットマークがある場合は削除する。

## 操作仕様

- `Cmd/Ctrl + Z`: Undo
- `Cmd/Ctrl + Shift + Z` または `Cmd/Ctrl + Y`: Redo
- `Delete` または `Backspace`: 選択中アイテムを削除
- `Escape`: ドラフト操作を取消し、選択ツールへ戻る
- `Enter`: 配線ドラフトを確定
- `R`: 単体選択中の部品を 90 度回転

Undo / Redo はキーリピートでは連続実行しない。

## 保存仕様

### 自動保存

- 保存先は `localStorage`。
- キーは `universal-board-layout-v1`。
- 操作のたびに現在状態を保存する。

### JSON 書き出し

現在の `board` と `items` を JSON として書き出す。

### JSON 読み込み

JSON を読み込み、正規化して現在レイアウトに反映する。

不正な JSON は読み込まず、アラートを表示する。

### PNG 書き出し

現在キャンバスに表示されている内容を PNG として書き出す。

## データ形式

### ルート

```json
{
  "board": {
    "cols": 30,
    "rows": 20,
    "pitch": 2.54,
    "showLabels": true,
    "theme": "green",
    "disabledHoles": ["4,7", "5,7"]
  },
  "items": []
}
```

`theme` は `green`、`brown`、`blue`、`black`。`disabledHoles` は無効穴の `x,y` キー配列。

### 配線

```json
{
  "id": "wire-...",
  "type": "wire",
  "layer": "top",
  "color": "#276ef1",
  "width": 2,
  "points": [
    { "x": 1, "y": 1 },
    { "x": 5, "y": 1 }
  ]
}
```

`layer` は `top` または `bottom`。

### 抵抗

```json
{
  "id": "resistor-...",
  "type": "resistor",
  "x": 5,
  "y": 5,
  "rotation": 0,
  "name": "R1",
  "value": "10k",
  "color": "#d7aa55",
  "style": "carbon",
  "length": 3
}
```

### コンデンサ / LED

```json
{
  "id": "capacitor-...",
  "type": "capacitor",
  "x": 5,
  "y": 5,
  "rotation": 90,
  "name": "C1",
  "value": "100n",
  "color": "#2f74c0",
  "style": "ceramic",
  "spacing": 3
}
```

抵抗の `style` は `carbon`、`metalBlue`、`metalGreen`。コンデンサの `style` は `ceramic`、`electrolytic`、`film`。

LED も同じ構造で、`type` が `led` になる。

### DIP IC

```json
{
  "id": "ic-...",
  "type": "ic",
  "x": 9,
  "y": 6,
  "rotation": 0,
  "name": "U1",
  "value": "NE555",
  "color": "#222831",
  "pins": 8,
  "width": 3
}
```

### ピンヘッダ

```json
{
  "id": "header-...",
  "type": "header",
  "x": 3,
  "y": 14,
  "rotation": 0,
  "name": "J1",
  "value": "PWR",
  "color": "#3b4754",
  "count": 3
}
```

### ラベル

```json
{
  "id": "label-...",
  "type": "label",
  "x": 3,
  "y": 13,
  "rotation": 0,
  "name": "L1",
  "value": "",
  "color": "#17202a",
  "text": "+  G  OUT"
}
```

### カットマーク

```json
{
  "id": "cut-...",
  "type": "cut",
  "x": 16,
  "y": 9
}
```

## 履歴仕様

- 変更前の状態を履歴スタックへ保存する。
- 最大履歴数は 80。
- Undo 時は現在状態を Redo スタックへ移し、履歴から直前状態を復元する。
- Redo 時は現在状態を Undo スタックへ移し、Redo スタックから状態を復元する。
- グループ移動は 1 手として扱う。

## 描画仕様

- キャンバスは画面サイズと device pixel ratio に応じてリサイズする。
- 穴は銅ランドと中心穴で描画する。
- 選択中アイテムには白と青の破線アウトラインを表示する。
- 複数選択では各アイテムに選択アウトラインを表示する。
- 配線は色線の下に白い縁を描画し、黒い IC や濃い部品の上でも視認できるようにする。
- 配置ドラフト中はゴースト部品、開始穴、終了穴、穴間隔を表示する。
- 範囲選択中は半透明の青い選択矩形を表示する。

## 現時点の制限

- 部品同士の干渉チェックは行わない。
- 回路図からの電気的な正誤チェックは行わない。
- 配線の自動ルーティングは行わない。
- 部品ライブラリは固定。
- IC はクリック配置のみで、ドラッグによる範囲決定には未対応。
- JSON スキーマファイルは未作成。
